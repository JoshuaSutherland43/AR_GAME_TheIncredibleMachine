# 🧠 AR Incredible Machine

An Augmented Reality reimagining of the classic *The Incredible
Machine*, built using marker-based AR to transform puzzle-solving into a
physical, spatial experience.

## 📌 Overview

This project recreates the core idea of *The Incredible Machine*---a
physics-based puzzle game---within an AR environment. Players construct
solutions by placing and manipulating objects in real-world space, then
running a simulation to test their design.

Unlike traditional versions, this implementation makes **AR a core
gameplay mechanic**, not just a visual enhancement.

## 🎮 Features

-   📍 Marker-Based AR Environment\
-   🧱 Interactive Object Placement\
-   🔄 Advanced Object Manipulation\
-   🎯 Physics-Based Simulation\
-   🧭 Trajectory Assistance\
-   🔁 Iterative Gameplay Loop\
-   🔘 AR-Based Start Mechanic\
-   🔊 Audio Feedback\
-   📐 Grid-Based Stability System

## 🕹️ How to Play

1.  Scan image targets to load environment\
2.  Tap to select and move objects using your phone\
3.  Tilt for depth control\
4.  Shake to rotate objects\
5.  Look at the AR button to start simulation\
6.  Adjust and retry until successful

## 🧩 Core Mechanics

-   Object placement and spatial manipulation\
-   Physics simulation via AR interaction\
-   Win condition: ball reaches goal\
-   Persistent objects with reset loop

## 🛠️ Technology Stack

-   Unity\
-   Vuforia Engine\
-   C#

## 🚧 Limitations

-   Requires image targets\
-   Rotation precision limitations\
-   Lighting affects tracking
